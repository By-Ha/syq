## Sucking YQTB

A simple program for the fucking silly epidemic reporting system.

### Usage

Rename `at_school.example.toml` or `other.example.toml` to `config.toml` and modify it with your information.

```bash
./syq -c [JSESSIONID] (-r)
```

For more information, please use:

```bash
./syq -h
```
